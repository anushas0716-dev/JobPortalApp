# TalentBridge – Job Portal Auth UI

A polished React registration & login UI for a Job Portal application.

## Features
- Login with email/password, remember me, forgot password flow
- Google & LinkedIn social login buttons
- 4-step registration wizard (Role → Info → Password → OTP)
- Live password strength meter
- 6-digit OTP input with auto-focus
- Responsive dark theme with amber accents

## Getting Started

```bash
npm install
npm start
```

Opens at http://localhost:3000
